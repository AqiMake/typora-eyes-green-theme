# Eyes Green

本主题基于Typora默认主题 [GitHub](https://theme.typora.io/theme/Github/) 开发，在该主题的基础上调整了背景颜色，并对部分markdown显示效果做了调整，旨在为浏览或编辑markdown文件时提供舒适护眼的使用体验。

该主题十分纯净、简单，在保证默认主题的性能、使用效率且不改变使用习惯的同时提供了良好的护眼体验。

## 说明

在 Windows 11 上设计和测试。还没完全测试，但应该能在 MacOs/Linux 上运行。

## 安装
1. 点击本页面的“下载”按钮，下载文件并解压。

2. 在 Typora 中打开“偏好设置”（文件 > 偏好设置）。切换至“外观”选项卡，点击“打开主题文件夹”按钮。

3. 将 `eyes-green.css` 文件复制到 Typora 的主题文件夹中。

4. 重启 Typora，从主题菜单中选择 Eyes Green 主题。
